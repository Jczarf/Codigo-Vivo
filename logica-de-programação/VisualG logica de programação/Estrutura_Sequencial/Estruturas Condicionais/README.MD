## Estrutura condicional Composta:  

| Operador | Significado       |
| -------- | ----------------- |
| >        | Maior             |
| <        | Menor             |
| >=       | Maior ou Igual    |
| <=       | Menor ou Igual    |
| =        | Igual             |
| <>       | Diferente         |

Exemplos  : 

(Suponha que N é 15)
 
| N sendo 15 |                   |
| -------- | ----------------- |
| N > 0    | Resultado: V      |
| N = 6    | Resultado: F      |
| N <= 50  | Resultado: V      |
| N <> 12  | Resultado: V      |
